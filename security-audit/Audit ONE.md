# Audit ONE

## Activity Overview

In part one of this activity, you will conduct an internal security audit, which you can include in your cybersecurity portfolio. Audits help ensure that security checks are made to monitor for threats, risks, or vulnerabilities that can affect an organization's business continuity and critical assets.
